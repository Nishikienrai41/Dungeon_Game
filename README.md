# Dungeon Game

Dit is een roguelike dungeon game gemaakt met Godot.

## Hoe het project openen

1. Download Godot Engine:
   Ga naar: https://godotengine.org/download  
   Download de juiste versie (bij voorkeur dezelfde versie als gebruikt in dit project, bijvoorbeeld 4.x).

2. Pak het ZIP-bestand van dit project uit op je computer.

3. Open Godot.

4. Klik op **Import**.

5. Ga naar de uitgepakte projectmap en selecteer het bestand:
   project.godot

6. Klik op **Import & Edit**.

7. Druk op **F5** of klik op **Run** om het spel te starten.

## Besturing

- Pijltjestoetsen → Bewegen  
- Spatie → Wachten  

Veel plezier!
